# UpanScan

一个自动监控USB设备并扫描复制文件的Python工具。

## 说明

UpanScan 可以实时监控USB设备的插入和移除，自动扫描设备中的文件，并根据配置规则复制匹配的文件到指定目录。

## 功能特性

- 🔍 实时监控USB设备状态
- 📁 自动扫描USB设备文件结构  
- ⚙️ 支持黑白名单设备过滤
- 📊 记录每次扫描的详细统计信息
- 🔄 支持文件规则匹配和选择性复制
- ⏱️ 可配置的复制速率限制
- 🧵 支持多线程文件复制
- 📝 完整的操作日志记录

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python main.py
```

## 配置说明

在运行前，请根据需求修改代码中的配置区域：

- **工作目录**：设置程序运行的工作路径
- **保存路径**：设置扫描结果的保存位置
- **设备过滤**：配置USB设备的黑白名单
- **扫描深度**：设置目录扫描的层级深度
- **复制设置**：配置文件复制速率和线程数
- **匹配规则**：定义需要复制的文件筛选条件

## 项目结构

```
UpanScan/
├── SAVE/
│   └── files/           # 复制的文件存储目录
├── files_records/       # 文件记录存储目录
├── UpanScan_logs.log    # 程序运行日志
└── {序列号}_records.json # 设备扫描记录
```
