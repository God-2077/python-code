## 文件类型检测

一个简单的文件类型检测工具

## 说明

下载最新版

安装依赖

```bash
pip install -r requirements.txt
```

使用方法

```bash
python main.py <file_path>
# 示例
python main.py example.txt
```

帮助

```bash
python main.py -h
```

| 选项 | 描述 |
| --- | --- |
| path | 自动检测文件或目录 |
| -h, --help | 显示帮助信息 |
| -f FILE, --file FILE | 分析单个文件 |
| -d DIR, --dir DIR | 分析整个目录 |
