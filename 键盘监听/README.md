## 键盘监听

键盘监听

最新版：[v.24.06.10.py][1]

## 说明

下载最新版

安装第三方库

```bash
pip install -r requirements.txt
```

运行

```
python ***.py
```


config.ini

```
[config]
file=./ada.txt
wait=aw4
waittime=10
exit=aaa
```

file 为键盘监听按键记录保存的路径
输入aw4等待10秒
输入aaa退出
