## 键盘监听

键盘监听

最新版：[Keyboard monitoring][1]

## 说明

下载最新版

安装第三方库

```bash
pip install keyboard && pip install ConfigParser
```

运行

```
python ***.py
```


config.ini

```
[config]
file=./keyboard.txt
```
file 为键盘监听按键记录保存的路径
