## 键盘监听

键盘监听

最新版：[v.24.06.16.py][1]

## 说明

下载最新版

安装第三方库

```bash
pip install -r requirements.txt
```

运行

```bash
python ***.py
```


config.ini

```ini
[config]
file=./ada.txt
wait=aw4
waittime=10
exit=aaa
keynumber=6
```

file 为键盘监听按键记录保存的路径
输入 aw4 等待10秒
输入 aaa 退出
连续输入 6 个数字时记录
